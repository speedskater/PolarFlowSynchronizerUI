Polar Flow Synchronizer UI

A GUI application for synchronizing your Polar Flow trainings to a local directory.

Welcome to Polar Flow Synchronizer UI

The Polar Flow Synchronizer UI can be used to synchronize all your trainings tcx-files stored in the Polar Flow web portal to a local directory.

Downloads

Download Windows Download OsX Download Linux

Usage

The Usage is as follows:

  1. Download the Application for (Windows, Mac Os, Linux)
  2. Install The Application
  3. Enter Username, Password and Choose your Export Directory
  4. Press the 'Sync' Button
